# OOPSBANNERAPP

This repository contains the implementation of an object-oriented programming system (OOPS) banner application. The project is divided into multiple use cases, each implemented in separate Java files:

- **uc1.java**: Handles the first use case.
- **uc2.java**: Implements the second use case.
- **uc3.java**: Covers the third use case.
- **uc4.java**: Manages the fourth use case.

## Branch Structure

- **main**: Contains only this README file.
- **dev**: Contains the latest code for all use cases.
- **features/uc1**: Development branch for the first use case.
- **features/uc2**: Development branch for the second use case.
- **features/uc3**: Development branch for the third use case.
- **features/uc4**: Development branch for the fourth use case.
